# 요구사항 프롬프트

```
아름다운 타워 디펜스 게임을 다음 사양으로 만들어주세요:

🎮 게임 컨셉:
- 판타지 중세풍 타워 디펜스
- 적들이 왼쪽에서 오른쪽으로 구불구불한 길을 따라 이동
- 플레이어가 타워를 배치해서 적들의 파도를 막아냄
- 깔끔하고 화려하며 매력적인 비주얼 스타일

🎨 비주얼 스타일:
- 밝은 카툰/판타지 아트 스타일
- 생생한 색상 (풀밭은 초록, 길은 갈색, 마법은 파랑)
- 부드러운 애니메이션과 파티클 효과
- 중세 판타지 테마의 UI

🏗️ 핵심 기능:
1. 격자 기반 타워 배치 시스템
2. 미리 정의된 적 경로 (S자 곡선이나 지그재그)
3. 3가지 타워: 궁수(빠름), 대포(범위공격), 마법(느림/빙결)
4. 3가지 적: 빠른 정찰병, 중장갑 기사, 날아다니는 드래곤
5. 난이도가 증가하는 웨이브 시스템
6. 자원 관리 (적을 죽여서 골드 획득)
7. 타워 업그레이드 시스템 (각각 3단계)

🎯 게임 메커니즘:
- 빈 격자 공간을 클릭해서 타워 배치
- 적들이 웨이브 단위로 등장하며 사이에 휴식시간 있음
- 각 적 타입마다 다른 체력, 속도, 골드 보상
- 타워들이 사정거리 내 가장 가까운 적을 자동 타겟
- 적들이 끝까지 도달하면 플레이어 패배
- 생존한 웨이브 수 + 남은 골드로 점수 계산

📱 기술적 요구사항:
- HTML5 캔버스 또는 모던 웹 프레임워크
- 데스크톱/모바일 반응형 디자인
- 부드러운 60fps 애니메이션
- 공격과 폭발을 위한 파티클 효과
- 점진적 난이도 증가
- 로컬 최고점수 저장

🎵 분위기:
- 중세 판타지 사운드트랙
- 공격과 폭발의 만족스러운 효과음
- 마법의 파티클 궤적과 스펠 이펙트
- 부드러운 카메라와 UI 전환

시각적으로 놀랍고 많은 정교함과 재미요소를 넣어서 만들어주세요!

# 레벨 디자인 프롬프트

타워 디펜스 맵 레이아웃을 디자인해주세요:
- 15x10 격자 시스템
- 왼쪽 끝에서 오른쪽 끝으로 이어지는 구불구불한 길
- 길이 최소 2-3번은 꺾여야 함
- 타워 배치를 위한 전략적 위치 확보
- 장식 요소 추가 (나무, 바위, 꽃)
- 균형잡힌 난이도 (너무 쉽지도 어렵지도 않게 방어 가능)

# 비주얼 디자인 프롬프트

다음 아트 스타일로 타워 디펜스 게임 에셋을 만들어주세요:
- 카툰 판타지 스타일, 밝고 화려함
- 타워들: 중세 궁수 타워, 돌 대포 타워, 수정 마법 타워
- 적들: 초록 고블린 정찰병, 중장갑 오크 기사, 빨간 드래곤
- 환경: 풀밭 타일, 돌길, 장식용 자연 요소들
- UI: 중세풍 나무와 금속 프레임, 판타지 폰트
- 이펙트: 반짝임, 폭발, 마법 궤적, 데미지 숫자

# 게임 밸런싱 프롬프트

이 타워 디펜스 게임의 밸런스를 맞춰주세요:

타워들:
- 궁수: 빠른 공격, 낮은 데미지, 저렴함 (10 골드)
- 대포: 느린 공격, 높은 범위 데미지, 비쌈 (25 골드)
- 마법: 중간 공격, 슬로우 효과, 중간 가격 (20 골드)

적들:
- 정찰병: 빠름, 낮은 체력 (20), 2 골드 가치
- 기사: 느림, 높은 체력 (80), 5 골드 가치
- 드래곤: 중간 속도, 중간 체력 (50), 비행, 8 골드 가치

웨이브:
- 정찰병 5마리로 시작
- 점진적으로 적 수 증가
- 3웨이브마다 새로운 적 타입 등장
- 6웨이브 이후부터 혼합 적 웨이브

# 사운드 디자인 프롬프트

판타지 타워 디펜스 게임용 오디오를 만들어주세요:
- 배경음악: 경쾌한 중세 판타지 오케스트라
- 효과음: 활시위 소리, 대포 폭발, 마법 번개, 적 죽음, 코인 수집
- UI 사운드: 버튼 클릭, 타워 배치, 업그레이드 차임벨
- 환경음: 바람, 새 소리, 마법적 분위기
- 모든 사운드는 만족스럽고 반복적이지 않아야 함

# 단계별 프롬프트

## 기본 구조

타워 디펜스 게임의 기초를 만들어주세요:
- 15x10 격자가 있는 HTML5 캔버스 설정
- 왼쪽에서 오른쪽으로 이어지는 구불구불한 길 생성
- 기본 타워 배치 시스템 구현
- 길을 따라 이동하는 간단한 적 추가
- 기본 UI (골드 카운터, 웨이브 표시기) 포함

## 전투 시스템

전투 메커니즘을 추가해주세요:
- 타워 타겟팅 시스템 (사정거리 내 가장 가까운 적)
- 이동시간이 있는 투사체 시스템
- 적 체력과 데미지 시스템
- 죽음 애니메이션과 골드 보상
- 타워 배치 시 사정거리 표시

## 비주얼 폴리시

시각적 경험을 다듬어주세요:
- 공격과 죽음을 위한 파티클 효과
- 모든 게임 오브젝트의 부드러운 애니메이션
- 데미지 숫자 팝업
- 타워와 적 스프라이트 애니메이션
- 배경 장식과 분위기

## 게임플레이 확장

게임 경험을 완성해주세요:
- 다양한 타워와 적 타입들
- 웨이브 진행 시스템
- 타워 업그레이드 메커니즘
- 게임 오버와 승리 조건
- 최고점수 시스템과 재시작 기능
```
